Project Title:

Email Spam Classifier using Artificial Intelligence

Project Scope:

This project uses Artificial Intelligence and Machine Learning techniques to classify emails as spam or not spam (ham) based on the text content of messages.

Technologies Used:

• Python
• Google Colab
• Scikit-learn
• Pandas
• NumPy

Dataset:

A sample email dataset containing labeled messages (spam / ham) was used.

The dataset includes:

•Email text messages
•Labels (spam or ham)

Methodology:
1.Load the email dataset.
2.Convert text data into numerical features using CountVectorizer.
3.Split the dataset into training and testing sets.
4.Train a Naive Bayes Classifier model.
5.Predict whether an email is spam or not spam.
6.Evaluate model performance using accuracy and classification report.

Output:

• Accuracy: 100.0%
• Classification Report: Precision, Recall, and F1-score = 1.00
• Predicted Result: Spam
